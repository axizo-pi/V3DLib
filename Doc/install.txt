- Fix for Pi1 - From Issues.md 

Call to `qpu_enable()` in `Mailbox.cpp` fails. To fix:

sudo vi  /boot/firmware/config.txt:

```
# Enable DRM VC4 V3D driver on top of the dispmanx display stack
dtoverlay=vc4-fkms-v3d
max_framebuffers=2
```

Replace with:

```
[all]
#dtoverlay=vc4-fkms-v3d
max_framebuffers=2
gpu_mem=128
```

`dtoverlay` should *not* be specified for Pi1. In addition, this sets the memory split with `gpu_mem`.
